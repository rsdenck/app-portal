# Portal do Cliente
Diretório contendo todos os arquivos voltados para o autoatendimento do cliente.

## Instalação e Configuração
1. Certifique-se de que o PHP 8.x está instalado.
2. Os arquivos dependem do diretório ../includes/ para bootstrap e conexão com o banco.

## Principais Arquivos
- cliente_chamado.php: Listagem de chamados do cliente.
- cliente_abrir_ticket.php: Formulário de abertura de novos tickets.
- cliente_login.php: Tela de login do cliente.
